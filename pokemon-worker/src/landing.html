Fast Run

Chạy script trực tiếp không qua menu. Chọn script → bấm Run → tất cả máy chạy song song trong 2 giây.

🔴

Kết nối USB

Cắm cáp là nhận. Không cần WiFi, không lag, không mất kết nối. Hỗ trợ 16+ thiết bị cùng lúc.

🌿

Xem màn hình Live

VNC viewer tích hợp ngay trong app. Xem và điều khiển từng iPhone mà không cần mở thêm phần mềm.

💧

Quản lý Config

Sửa LICENSE_KEY, MAIL_SERVER và mọi config trực tiếp từ app. Áp dụng hàng loạt cho nhiều máy.

🔮

Bảo mật dữ liệu

Dữ liệu người dùng được mã hóa end-to-end. Mỗi thiết bị có key riêng, không ai truy cập được ngoài chủ sở hữu.

🔥

Log & Monitor

Stream log realtime từ mỗi thiết bị. Biết ngay máy nào lỗi, máy nào đang chạy, máy nào offline.

Cài PokemonFleet

Tải app từ link admin gửi. Cài trên Windows hoặc Mac. Không cần cấu hình phức tạp.

Cắm iPhone

Kết nối iPhone đã jailbreak qua USB. Bấm Trust trên điện thoại. App tự nhận thiết bị.

Setup & Config

Bấm Setup để cài POKEIOSControl lên iPhone. Điền LICENSE_KEY và MAIL_SERVER vào Config.

Chạy Script

Chọn script, tick máy cần chạy, bấm Run. Xong! Ngồi xem log hoặc đi uống cà phê.
